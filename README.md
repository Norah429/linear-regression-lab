# Linear Regression Lab

## Description
This project applies Linear Regression to predict customers' yearly spending using the Ecommerce Customers dataset.

## Steps
- Data exploration
- Data visualization
- Model training
- Prediction and evaluation

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
